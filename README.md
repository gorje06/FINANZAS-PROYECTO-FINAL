# FINANZAS-PROYECTO-FINAL

Simulador web **FinanCuota** — crédito vehicular, método francés, Compra Inteligente, VAN/TIR/TCEA.

Curso: Finanzas e Ingeniería Económica — UPC

## Arranque local (recomendado — BD permanente)

Doble clic en **`run-local.bat`** o en PowerShell:

```powershell
.\run-local.ps1
```

Abre: http://127.0.0.1:5000

| Admin | Valor |
|-------|-------|
| Usuario | `admin` |
| Contraseña | `adminupc` |

La base de datos SQLite se guarda en `financuota.db` (no se borra al cerrar).

## Subir a Render sin tarjeta

Lee el archivo **`INSTRUCCIONES_RENDER.txt`** (paso a paso, copiar y pegar).

Repo: https://github.com/gorje06/FINANZAS-PROYECTO-FINAL

## Stack

- Python / Flask
- SQLite (sin cambio de motor)
- HTML / CSS / JavaScript
